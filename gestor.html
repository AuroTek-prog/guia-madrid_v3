¿Cómo añadir un nuevo apartamento?

  1. Crea una carpeta nueva: assets/images/apartments/nuevo-slug/
  2. Sube las fotos:
    • portada.jpg (foto principal del encabezado)
    • Opcional: acceso.jpg (puerta o caja fuerte)
  3. Edita data/apartments.json:
    • Copia un bloque existente (ej. "sol-101")
    • Cambia la clave a algo único: "nuevo-slug"
    • Actualiza "name", "address", "zone"
    • Actualiza "images.portada" con la nueva ruta
    • Si usas Raixer: pon "raixerDevices.deviceId" con el ID real de este piso
  4. Commit y push → ¡la guía y esta página se actualizan automáticamente!

Si necesitas ayuda con fotos o datos, envíamelos y lo subo yo.
